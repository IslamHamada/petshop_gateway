# API Gateway

Central entry point for all PetShop services.

## Description
Implements routing, JWT validation, CORS configuration, and request forwarding.  
Handles Auth0 access tokens and exposes the unified API.

## Tech Stack
- Java 17  
- Spring Boot 3.x  
- Spring Cloud Gateway
